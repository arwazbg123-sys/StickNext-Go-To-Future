🎮 CARA MEMBUKA:
====================

✅ CARA PALING MUDAH (Langsung buka file):
   1. Buka folder: StickNext Go To Future Wikies\wiki\
   2. Double-click file: index.html
   3. Browser akan terbuka otomatis
   ✨ Wiki sudah FULLY RESPONSIVE di semua ukuran layar!

⚡ CARA LEBIH BAIK (Menggunakan Server Lokal):

   📌 Windows dengan Python:
      1. Buka Command Prompt (atau PowerShell)
      2. Navigasi ke folder wiki:
         cd "C:\Users\Naufal Maulana Rizki\Documents\StickNext Go To Future Wikies\wiki"
      3. Jalankan perintah:
         python -m http.server 8000
      4. Buka browser → http://localhost:8000

   📌 Alternative dengan Node.js (Lebih cepat):
      1. Install global http-server:
         npm install -g http-server
      2. Navigasi ke folder wiki dan jalankan:
         http-server .
      3. Akses di browser sesuai link yang ditampilkan

   📌 VS Code (Live Server Extension):
      1. Install extension "Live Server"
      2. Right-click file index.html
      3. Pilih "Open with Live Server"

Jika ingin menambah fitur lebih lanjut, bisa dengan:

   - Edit styling di assets/css/styles.css

   - Tambah data di assets/js/data.js

   - Buat file app.py untuk server
   - Setup Flask untuk REST API
   - Connect ke database MySQL/PostgreSQL
   - Contoh sudah di README.md

   - Gallery gambar
   - Video embeds
   - User authentication
   - Admin panel
   - Forum/comment system

💡 TIPS & TRICKS:
==================

🔹 Untuk mengedit konten:
   - Data ada di: assets/js/data.js
   - Ubah isi object wikiData untuk update
   - Auto-refresh di browser

🔹 Untuk tambah halaman baru:
   1. Tambah di wikiData di data.js
   2. Buat function render di app.js
   3. Tambah link di navigation

🔹 Debug di browser:
   - Tekan F12 untuk buka DevTools
   - Lihat Console untuk error messages
   - Gunakan Network tab untuk monitor requests

🔹 Testing mobile responsiveness:
   - Tekan Ctrl+Shift+M (atau Cmd+Shift+M di Mac)
   - DevTools akan enter responsive design mode
   - Pilih device dari dropdown (iPhone, iPad, Android, custom)
   - Lihat MOBILE_TESTING_GUIDE.md untuk checklist lengkap

🔹 Hamburger menu:
   - Otomatis muncul di mobile (max 1023px width)
   - Click icon untuk toggle
   - Smooth animation
   - Auto-close saat klik link

   - Auto-fit grid
   - Compact padding

   - Menu masih horizontal, compact
   - 2-column grid
   - Medium padding

   - Hamburger menu ☰

   - Touch-friendly (44px min)
   - Small fonts tapi readable

✅ Small Phone (320px - 480px) **HAMBURGER MENU**
   - Hamburger menu ☰
   - 1-column grid super compact
   - Minimal padding
   - Min font 12-13px

✅ Tablet Landscape (768px+ width, landscape)
   - Menu horizontal
   - 2-column grid
   - Compact layout

✅ Phone Landscape (max 767px width, landscape) **HAMBURGER MENU**
   - Hamburger menu ☰
   - 1-column, minimal height
   - Max-height fixed menu
